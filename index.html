🖥️ Terminal Basics

  • date — Display current date and time.
  • echo "Hello World" — Print text to screen.
  • clear — Clear the terminal screen.
  • whoami — Show current username.
  • who — See who is logged into the system.
  • sort filename.txt — Sort the lines in a file.

🔗 Pipe Operator ( | )

Definition: Send output of one command into another command.

cat file.txt | sort

This sorts the contents of file.txt.

📜 Bash Scripting Essentials

🖊️ Variables

name="Linux"
echo "Hello, $name!"

🔢 Positional Parameters

echo "First arg: $1"
echo "Second arg: $2"

🛠️ For Loop

for i in 1 2 3
do
  echo "Item $i"
done

🔁 While Loop

count=1
while [ $count -le 3 ]
do
  echo "Number $count"
  ((count++))
done

🔀 Case Statement

read -p "Choose a number: " num
case $num in
  1) echo "One";;
  2) echo "Two";;
  *) echo "Other";;
esac

🧮 Arithmetic Operations

Addition, subtraction, multiplication, division in Bash:

a=10
b=5
echo "Sum = $((a+b))"
echo "Subtract = $((a-b))"
echo "Product = $((a/*b))"
echo "Division = $((a/b))"

🔧 Functions

greet() {
  echo "Welcome, $1!"
}
greet "User"

🌐 Shell Types

  • System Shell: Provided by OS (e.g., /bin/sh).
  • User Shell: Chosen by user (e.g., bash, zsh).

🔒 Predefined Shell Variables

  • $HOME — Home directory path.
  • $USER — Current user.
  • $PWD — Current working directory.
  • $SHELL — Shell program path.
  • $PATH — Search path for executable files.

🖊️ User-Defined Variables

Define your own variable:

myvar="Welcome"
echo "$myvar to Linux!"

⚙️ System Programming (C Language)

🔹 system() Function

Runs a shell command inside a C program.

#include <stdlib.h>
int main() {
    system("ls -l");
    return 0;
}

🔹 fork() - Create a New Process

Creates a child process.

#include <unistd.h>
#include <stdio.h>
int main() {
    fork();
    printf("Hello Process\n");
    return 0;
}

🔹 exec() - Replace Current Process

Runs a different program inside current process.

#include <unistd.h>
int main() {
    execl("/bin/ls", "ls", "-l", NULL);
    return 0;
}

🔹 wait() - Wait for Child

Parent waits for child process to finish.

#include <sys/wait.h>
#include <unistd.h>
int main() {
    fork();
    wait(NULL);
    return 0;
}

🔹 exit() - End a Program

#include <stdlib.h>
int main() {
    exit(0);
}

🔹 getpid() and getppid()

getpid() — Get current process ID, getppid() — Get parent process ID.

#include <unistd.h>
#include <stdio.h>
int main() {
    printf("My PID: %d\n", getpid());
    printf("My Parent PID: %d\n", getppid());
    return 0;
}
